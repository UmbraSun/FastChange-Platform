# FastChange Platform

## Project Goal

The FastChange Platform is a FinTech demo project created to showcase practical experience with a modern backend, frontend, and mobile technology stack during job interviews.
The project is not a commercial product but serves as an engineering case study to demonstrate architecture, technologies, and approaches.

---

## Main idea

The system models a currency exchange platform supporting:

- user wallets
- deposit and withdrawal operations
- currency exchange
- event-driven architecture
- integration with message brokers

---

## Key principles

- Clean Architecture
- CQRS (Command Query Responsibility Segregation)
- Event-Driven Architecture
- Explicit separation of layers
- No direct Infrastructure → Domain dependencies
- Extensive demonstration of modern technologies

## Project criteria

A project is considered successful if it demonstrates:

- an understanding of distributed systems
- work with message brokers
- implementation of Clean Architecture
- handling of transactions and data consistency
- integration of multiple technologies into a single system
